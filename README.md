# With Joyful Lips

A Progressive Web App for hymn music streaming and discovery. Worship, praise, and gospel music for every moment.

**Live**: [joyfullips.com](https://joyfullips.com)

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Material UI 7
- Progressive Web App

## Getting Started

### Prerequisites

- Node.js >= 18
- npm >= 9

### Setup

```bash
npm install
cp .env.example .env    # Fill in your environment variables
npm run dev
```

### Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Type-check + production build |
| `npm run preview` | Serve production build locally |
| `npm run lint` | Run linter |

## Environment Variables

Copy `.env.example` to `.env` and configure the required values. See the example file for details.

## Features

- Stream and discover hymns, worship, and gospel music
- Search and browse by categories
- Create and manage personal playlists (offline-capable)
- Multiple sign-in options
- Dark mode support
- Installable as a native-like app (PWA)
- Media session integration (lock screen controls)
